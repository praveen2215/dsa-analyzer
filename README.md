# DSA Analyzer

A professional LeetCode analytics dashboard built with React + Node.js.

## Features
- Real LeetCode data via GraphQL API
- Authentication with session-based login
- Difficulty breakdown, topic heatmap, rating progression
- Interview readiness score
- Skill gap analyzer (Google, Meta, Amazon, Microsoft, Apple, Netflix)
- Daily challenge tracker with streak
- AI study plan with weak topic detection
- Goal tracker (weekly/monthly)
- Activity heatmap, language breakdown
- Saved profiles and analysis history

## Tech Stack
- **Frontend:** React, Vite, Chart.js
- **Backend:** Node.js, Express, SQLite, Passport.js
- **Auth:** Session-based with bcrypt

## Setup

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173
